# Boss Battle Designs — Pokemon Platinum Blitz

All boss Pokemon are standardized with:
- **31 IVs** in all stats
- **Optimized EVs** (252/252/4 spreads unless noted)
- **Competitive held items** (see availability notes per gym)
- **4-move competitive sets**
- **Maximum AI difficulty flags**

## Design Notes

**Held item availability** is gated to what is realistically obtainable by each gym:
- Gyms 1–2: Basic berries only
- Gym 3: Salac/Sitrus berries
- Gym 4+: Life Orb, Choice items, Leftovers

**Level cap rationale:** Each cap is set to the highest-level Pokemon on the next boss's team, ensuring players can always reach parity but not exceed it before the fight.
